# VPlayer
根据规则抓取视频网站搜索结果中的视频信息,可播放可选集

### 目前很多小众视频网站广告繁杂,看视频很不方便,该项目旨在通过抓取网页搜索结果来实现清新的观看体验

#### 已实现功能
1. 完成3个网站的适配
2. 可直接播放
3. 可选集
4. 可记录已看视频位置(有些视频地址会动态变化,无法实现记录)
